# 创建
```
cd my_project
python3 -m venv venv
```

# 激活
windows
```
venv\Scripts\activate
```
mac/linux
```
source venv/bin/activate
```

# 安装依赖
```
pip install package_name
```

# 运行程序
```
python my_script.py
```

# 退出
```
deactivate
```
