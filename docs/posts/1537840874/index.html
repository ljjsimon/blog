大多数现在语言都有自己的包管理器，比如 php 的 composer，和 nodejs 的 npm。但是如果语言没有包管理器，或者想要的库没有打包，又不想手动更新，该怎么办呢？答案是 git 子模块。下面来一步一步建一个子模块。

这时目录下会多出我们的子模块 php-view 和一个 .gitmodules 文件，这个文件里保存了子模块的映射关系。这个操作在子目录下也可行。
